# Météo & Trafic Aérien - EBLG/EBCI

Ce projet affiche en temps réel :
- La météo aux aéroports de Liège (EBLG) et Charleroi (EBCI)
- Le sens de piste en vigueur
- La liste des vols confirmés
- Les données des sonomètres fixes

## Prérequis
- Clés API pour la météo et les données aériennes
- Un serveur web pour héberger le projet

## Installation
1. Clone ce dépôt
2. Remplace les clés API dans `script.js`
3. Déploie sur GitHub Pages ou un autre hébergeur
